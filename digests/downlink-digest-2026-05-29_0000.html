Overview

The North Korean state-sponsored threat actor Kimsuky (also known as Velvet Chollima) was observed conducting a fresh wave of cyber attacks against South Korean military and corporate entities during March and April 2026. The operations employed tailored social engineering, multi-stage payload delivery, and the abuse of legitimate tools, while introducing new malware variants and verification techniques. The group’s activities were analyzed by ENKI, Kaspersky, CrowdStrike, Darktrace, and Logpresso, revealing an expanding arsenal and a continued focus on sensitive data theft.

Campaign Details

March 2026: Fake Security Software Installers

  • A bogus web page impersonated the security software installation page of a South Korean B2B messaging service.
  • The lure likely targeted messaging administrators within corporate environments.
  • Two executables, “nos-setup.exe” and “astx-setup.exe”, masqueraded as nProtect Online Security and AhnLab Safe Transaction (ASTx) but contained identical malicious code.
  • The binaries launched a second-stage DLL (“MemLoader.dll”) via regsvr32.exe, then deleted themselves via a batch script.
  • The DLL established persistence through a scheduled task and contacted a command-and-control (C2) server, where the attacker likely monitored GET requests and selectively delivered payloads to specific victims (the exact payload remains unknown).

April 2026: Counterfeit Cisco Webex Page

  • A counterfeit Webex page displayed a pop-up urging victims to download and run a script to resolve camera access issues.
  • This resulted in downloading a ZIP archive containing an encrypted JavaScript file (“fix-camera.jse”).
  • Execution of the JSE deployed an intermediate downloader (“mTSTCv8.mdxm”) via PowerShell, which performed anti-analysis checks and fetched the next stage (“engine.dat” or “spyInster.dll”).
  • The DLL dropped a loader (“cacheMon.dat”) that executed HTTPSpy on the system.
  • HTTPSpy is a full-featured remote access trojan capable of running shell commands, uploading/downloading files, launching processes, capturing screenshots, injecting DLLs into specified processes, and self-erasing.
  • The malware also dropped and opened “meeting.html”, which redirected the victim to a real Webex meeting room associated with an event scheduled around the same time, indicating the attacker likely compromised a previous attendee to steal the schedule and then engineered a fake page to distribute malware to other participants.

Key Malware Families and New Variants

The attack chains dropped a variety of loaders written in JSE, PIF, SCR, and EXE formats, delivering malware from two broad clusters: PebbleDash and AppleSeed.

  • HelloDoor – A Rust-based PebbleDash variant first seen in August 2025, likely developed with the help of a large language model (LLM). Provides basic features: set current directory, sleep, and execute commands.
  • HttpMalice – The latest PebbleDash backdoor variant, active by December 2025. Gathers system information, sets persistence, performs reconnaissance with native Windows commands, captures screenshots, loads downloaded payloads into memory, runs commands, and exfiltrates output.
  • HttpTroy – A backdoor delivered via a loader named MemLoad; supports file transfer, screenshot capture, command execution, in-memory loading of executables, reverse shell, process termination, and trace removal.
  • AppleSeed – Comes as a Dropper (downloads additional malware, executes commands) and a Spy variant that collects documents, screenshots, keystrokes, USB drive listings, and specifically harvests data from the C:\GPKI directory (mirroring the earlier Troll Stealer).
  • HappyDoor – An advanced version of AppleSeed first identified in 2021.

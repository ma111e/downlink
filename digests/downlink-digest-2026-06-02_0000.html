01 Active Exploitation: Android CVE-2025-48595 and Oracle WebLogic CVE-2024-21182

Google’s June 2026 Android security update addressed 124 vulnerabilities, including CVE-2025-48595, a high-severity Framework privilege escalation flaw reportedly under limited, targeted exploitation. The integer overflow issue affects Android 14, 15, 16, and 16 QPR2 and may allow local escalation of privilege and code execution without user interaction or additional execution privileges. Google released 2026-06-01 and 2026-06-05 patch levels, with the latter covering kernel and third-party chipset components from vendors including Qualcomm, MediaTek, Unisoc, and Imagination Technologies.

CISA added Oracle WebLogic Server CVE-2024-21182 to its Known Exploited Vulnerabilities catalog following evidence of active exploitation. The flaw, patched by Oracle in July 2024, affects WebLogic Server versions 12.2.1.4.0 and 14.1.1.0.0 and can be exploited by an unauthenticated network attacker via T3 or IIOP to compromise affected servers or access critical data. Reporting noted more than 1,500 internet-exposed WebLogic servers running versions identified as vulnerable.

02 Malware Campaigns Targeting Gamers, macOS Users, and Developer Supply Chains

The WeedHack campaign has infected 116,464 systems since January through malicious Minecraft mods, clients, cheats, and utilities promoted via YouTube and SEO poisoning. McAfee identified more than 240 distribution URLs and 3,820 unique malicious JAR files tied to the campaign. The stealer targets Minecraft session IDs, browser cookies and saved passwords, cryptocurrency wallets, Discord, Steam, Telegram credentials, and screenshots, while a premium tier adds remote control, webcam access, keylogging, shell access, and file management.

Unit 42 reported Operation FlutterBridge, a macOS malvertising campaign delivering FlutterShell, a Flutter-based backdoor masquerading as applications such as PodcastsLounge, PDF-Brain, and PDF-Ninja. Observed samples were signed with valid Apple Developer IDs and passed notarization, and the malware uses a WebView JavaScript-to-native bridge to let operators modify malicious logic hosted externally. Some variants also include AI summarization functionality that sends document content to attacker-controlled infrastructure before forwarding requests to an AI agent.

Red Hat removed 32 packages from its distribution pipeline after a compromised GitHub account was used to distribute credential-stealing malware to developers. The packages were downloaded roughly 117,000 times per week, and the malware was described as a variant of the Mini Shai-Hulud self-propagating worm, whose source code was published by TeamPCP in May. Researchers have not confirmed whether TeamPCP conducted the attack or whether another actor reused the released code.

03 Espionage and Government-Focused Intrusions

Sekoia attributed January 2026 Gamaredon activity against Ukraine to exploitation of WinRAR CVE-2025-8088, a path traversal flaw used to launch GammaPhish. The infection chain retrieves GammaLoad, which fingerprints hosts, updates registry network settings using dead drop resolvers, and fetches VBScript payloads. Delivered payloads include GammaWorm, which spreads through network shares and USB drives using malicious LNK files, and GammaSteel, a modular stealer that exfiltrates selected files to AWS S3 or fallback attacker infrastructure.

Researchers also disclosed Operation XENOFISCAL, a spear-phishing campaign likely conducted by the Pakistan-aligned SideCopy group against Afghanistan’s Ministry of Finance and related provincial offices. The campaign uses Pashto-language lure filenames, malicious LNK files, mshta.exe, a compromised Afghan education domain, and obfuscated in-memory JavaScript. The final payload is Xeno RAT 1.8.7, which supports file operations, keylogging, screenshots, clipboard monitoring, and webcam or microphone surveillance.

Russia’s FSB accused foreign intelligence services of conducting a large-scale mobile espionage operation against senior Russian officials. The agency alleged that malware was capable of data theft, communications interception, geolocation tracking, contact extraction, and covert audio and video surveillance, but did not name the spyware, describe infection vectors, or provide technical evidence.

04 AI, Browser Abuse, and Identity Platform Risks

Sophos reported a threat actor using an AI-built ransomware toolkit that automates Active Directory discovery and supports EDR evasion development. The toolkit included Cobalt Strike profiles, Telegram bot API command-and-control, Python shellcode injection scripts, a Cloudflare Worker redirector, and a Git repository containing an AD discovery panel plus a lab for testing against Sophos, CrowdStrike, and Windows Defender. Sophos said AI was used to accelerate development and testing, not embedded in deployed malware or operating autonomously in victim environments.

Other reporting focused on AI’s role in browser-layer threats and vulnerability operations. Push Security described phishing-as-a-service techniques such as ClickFix, InstallFix, ConsentFix, and device code phishing, while noting that short-lived domains, legitimate sites, malvertising, and search results weaken traditional indicator-based detections. A separate article argued that AI-driven exploitation is compressing disclosure-to-exploitation timelines and cited Anthropic’s Project Glasswing update, which said Claude Mythos Preview helped identify more than 10,000 high- or critical-severity vulnerabilities in one month.

Instagram users reported account takeovers after attackers allegedly abused Meta’s AI-powered recovery process with AI-generated selfie videos based on target profile photos. Some victims said the method bypassed two-factor authentication and changed account email addresses, affecting high-value handles including @hey, @korn, and an account previously used by the Obama White House team. Meta stated that the issue had been resolved and impacted accounts were being secured.

05 Service Disruption, Credential Security, and Wireless Exposure

Microsoft investigated a widespread Exchange Online incident affecting mail flow for customers in North America and Germany. The incident, tracked as EX1331830, caused significant delays and some undelivered messages, with users reporting temporary SMTP deferral errors tied to concurrent connection limits and abrupt connection closures.

Dashlane disclosed that fewer than 20 personal-plan users had encrypted vaults downloaded after a May 31 brute-force attack targeting account access, two-factor protections, and new device registration. The company said built-in controls triggered temporary suspensions and authentication issues, and that directly notified users were the only ones whose vaults were at risk.

Kaspersky GReAT published a passive wardriving assessment across Mexico City, Guadalajara, and Monterrey ahead of the 2026 FIFA World Cup. The study recorded 84,588 wireless signals and found widespread default or MAC-derived SSID naming, open networks representing roughly 10–12% of detected infrastructure, and WPS advertised by 45% of access points.

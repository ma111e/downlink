Article Overview and Discovery

The TrickMo Android banking malware has introduced a significant operational update tracked by ThreatFabric as Trickmo.C. Researchers have been monitoring this variant since January 2024, noting a shift in its distribution and communication methods. Originally identified in September 2019, the malware has remained in continuous development. Recent campaigns focus on users across France, Italy, and Austria. The malicious applications are carefully disguised as popular TikTok or streaming apps to bypass user suspicion. Prior analysis by Zimperium in October 2024 documented forty distinct variants distributed through sixteen droppers, communicating with twenty-two separate command-and-control infrastructures.

Command-and-Control Infrastructure

The most notable technical advancement in this variant is the integration of the TON blockchain for covert communications. Instead of relying on traditional domain-based servers, the malware utilizes .ADNL addresses routed through an embedded local TON proxy running directly on the infected device. This decentralized peer-to-peer network masks the operator’s IP addresses and communication ports behind 256-bit identifiers. Consequently, traditional domain takedown strategies are largely ineffective against this infrastructure. Network traffic analysis at the edge only reveals encrypted TON flows, which are completely indistinguishable from legitimate TON-enabled applications.

Operational Architecture and Capabilities

TrickMo operates using a modular, two-stage design. The initial host APK functions as a loader and persistence layer, while a runtime-downloaded APK module implements the offensive functionality. Once deployed, the malware executes a comprehensive suite of surveillance and data theft operations. These capabilities include:

  • Phishing overlays to capture banking credentials
  • Keylogging and live screen streaming
  • SMS interception and OTP notification suppression
  • Clipboard modification and notification filtering
  • Automated screenshot capturing

New Commands and Frameworks

ThreatFabric researchers have identified several new commands and capabilities added to the current variant. These additions expand the malware’s network reconnaissance and remote access potential:

  • Network tools: curl, dnsLookup, ping, telnet, and traceroute
  • Tunneling and proxying: SSH tunneling, remote port forwarding, local port forwarding, and authenticated SOCKS5 proxy support
  • Frameworks and permissions: The Pine runtime hooking framework is present but currently inactive due to a lack of installed hooks. Additionally, the malware declares extensive NFC permissions and reports NFC capabilities in telemetry, though no active NFC functionality has been observed.

Mitigation Recommendations

The article outlines specific defensive measures for Android users to reduce exposure to TrickMo and similar threats. Users are advised to exclusively download software from the official Google Play store. It is also recommended to limit the total number of installed applications on devices. Security best practices include using applications only from reputable publishers and ensuring that Play Protect remains active at all times.

EXEC Executive Overview

Threat intelligence reports surfaced during the coverage window detail significant evolution in the TrickMo Android banking malware campaign. Researchers have identified a new variant of the malware that has integrated the TON blockchain protocol to facilitate covert command-and-control (C2) communications. By leveraging blockchain infrastructure, the threat actors aim to obscure their command infrastructure from traditional domain-based takedown efforts and network monitoring mechanisms. This architectural shift represents a move toward decentralized communication channels to enhance the persistence and resilience of the malware’s backend.

The updated TrickMo variant maintains its operational focus on financial theft, specifically targeting banking credentials and cryptocurrency wallets. Geographically, the campaign has been observed targeting users in France, Italy, and Austria. To maximize installation success rates, the malware employs sophisticated obfuscation techniques, masquerading as legitimate applications such as TikTok or various streaming services. This disguise strategy is designed to bypass user scrutiny and potentially evade basic integrity checks based on application names or metadata.

Technically, the malware operates using a modular, two-stage design. The initial stage likely handles delivery and persistence, while the second stage deploys the functional modules responsible for malicious activities. Once active, the malware implements a comprehensive suite of espionage capabilities. These include phishing overlays that mimic legitimate banking interfaces to harvest credentials, keylogging to capture keystrokes, screen recording to monitor user activity, and SMS interception to bypass two-factor authentication mechanisms often used in financial transactions.

Analysis of the latest variant reveals the introduction of advanced operational commands that indicate a maturation of the malware’s remote access capabilities. Researchers have documented support for SSH tunneling, allowing the malware to create secure channels for subsequent commands. Additionally, the variant supports remote port forwarding and authenticated SOCKS5 proxy functionality. These features enable the malware to route traffic through compromised devices, potentially facilitating lateral movement or proxying malicious requests through the victim’s connection to mask the true origin of attacks.

The reporting from BleepingComputer highlights these technical advancements as evidence of an ongoing arms race between TrickMo developers and security researchers. The adoption of TON blockchain and network tunneling capabilities suggests a strategic pivot toward more resilient infrastructure and enhanced operational security for the threat actor group behind the campaign. This development underscores the increasing complexity of Android banking trojans and their ability to adopt emerging technologies for evasion and persistence.

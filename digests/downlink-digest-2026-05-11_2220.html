Concurrently, AI integration has accelerated attack velocity, collapsing vulnerability exploitation windows to approximately ten hours, while AI-driven threats include WebSocket hijacking in the Cline coding agent and a Chrome extension vulnerability compromising Anthropic’s Claude interface.

Critical vulnerability exploitation remains active across multiple platforms. Attackers are leveraging flaws in Ivanti Endpoint Manager Mobile and Palo Alto Networks PAN-OS firewalls to execute code with root privileges.

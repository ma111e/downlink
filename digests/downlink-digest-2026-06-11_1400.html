01 Calendar Phishing, OAuth Abuse, and Agentic AI Risk

Health-ISAC reported an emerging healthcare-sector social engineering pattern called CalPhishing, in which attackers use iCalendar files and enterprise calendar auto-processing to place malicious links directly into employee schedules. Recent campaigns reportedly use ConsentFix, also described as device code phishing, to capture session tokens and bypass multi-factor authentication protections after the user interaction stage.

Netskope Threat Labs framed OAuth authorization abuse as a structural scaling problem, particularly as organizations adopt AI agents that rely on persistent tokens, broad permissions, opaque runtimes, and autonomous workflows. The article described a confused deputy scenario in which an attacker plants malicious instructions in data consumed by an agent, causing the agent to use its own valid OAuth token to perform attacker-directed actions.

SpecterOps published research on automating indirect prompt injection testing against an agentic system with access to Notion search, shell execution, and web-fetching tools. The workflow used poisoned Notion content to progress from simple echo payloads to full-chain payloads involving command execution and HTTP exfiltration, while noting that newer Claude Sonnet model behavior detected some earlier payload styles more reliably.

02 Developer Supply Chain Exposure: Miasma, Hades, npm v12, and Socket

StepSecurity reported that the Miasma worm and its PyPI branch Hades are spreading across npm and PyPI ecosystems, with execution paths tied to project opening, package installation, or package import. Miasma uses a small binding.gyp file in a technique described as “Phantom Gyp,” while Hades relies on an obfuscated import hook in init.py, creating visibility gaps for tools focused primarily on install-time checks, process behavior, or declared dependencies.

Aikido highlighted npm v12, scheduled for July 2026, as a major package-manager security change because dependency install scripts will no longer run by default unless packages are allowlisted. The release is also expected to block Git dependencies and remote URL dependencies unless explicitly permitted, and to treat implicit node-gyp rebuilds triggered by binding.gyp as install scripts.

Socket announced Andrew Becherer as its first Chief Information Security Officer, positioning the appointment around enterprise concerns with open source, AI tooling, and malicious package risk. The company said it now protects more than 27,000 organizations and framed the role as both an internal security function and a customer-facing software supply chain security leadership position.

Law enforcement dismantled AudiA6, a cryptocurrency service Europol said was used by ransomware actors and other cybercriminals to launder more than $380 million. Authorities arrested two individuals in Georgia, seized domains, vehicles, properties, cryptocurrency assets, Telegram accounts, and retrieved 6,000 Know-Your-Customer records tied to alleged money mule accounts.

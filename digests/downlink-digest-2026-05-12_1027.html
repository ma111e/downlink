TeamPCP, the threat actor behind a recent supply chain attack spree, has compromised npm and PyPI packages from TanStack, UiPath, Mistral AI, OpenSearch, and Guardrails AI in a fresh Mini Shai-Hulud campaign.

The affected npm packages contain an obfuscated JavaScript file named router_init.js that profiles the execution environment and launches a credential stealer. This malware targets cloud providers, cryptocurrency wallets, AI tools, messaging apps, and CI systems like GitHub Actions. Exfiltrated data is sent to the filev2.getsession[.]org domain, a deliberate choice to evade enterprise detection due to its association with a privacy-focused messaging service.

As a fallback, encrypted data is committed to attacker-controlled GitHub repositories using stolen tokens via the GraphQL API. The malware also establishes persistence by injecting hooks into Claude Code and Visual Studio Code to survive reboots. Additionally, it installs a gh-token-monitor service to track and re-exfiltrate GitHub tokens while injecting malicious workflows that serialize repository secrets and upload them to api.masscan[.]cloud.

TanStack traced the compromise to a chained GitHub Actions attack involving pull_request_target triggers, cache poisoning, and runtime memory extraction of an OIDC token. Notably, no npm tokens were stolen, and the npm publish workflow itself remained uncompromised. The attackers staged the payload in a GitHub fork, injected it into published npm tarballs, and hijacked the legitimate TanStack/router workflow to publish compromised versions with valid SLSA provenance.

This makes the worm the first documented npm threat capable of producing malicious packages with valid SLSA Build Level 3 attestations. The worm spreads autonomously by finding npm tokens with bypass_2fa enabled, enumerating packages by the same maintainer, and exchanging GitHub OIDC tokens for per-package publish tokens. The TanStack compromise is assigned CVE-2026-45321 with a CVSS score of 9.6, impacting 42 packages and 84 versions.

Beyond npm, the campaign affected PyPI packages including guardrails-ai@0.10.1 and mistralai@2.4.6. Microsoft analysis revealed the mistralai package downloads a credential stealer with country-aware logic that avoids Russian-language environments. The payload carries a 1-in-6 chance of executing rm -rf / on systems detected in Israel or Iran.

The guardrails-ai compromise executes malicious code on import, checking for Linux systems before downloading and running a remote Python artifact without integrity verification. The campaign continues to propagate across search infrastructure, AI tooling, enterprise automation, and CI/CD ecosystems.

TeamPCP has executed a sophisticated Mini Shai-Hulud supply chain attack targeting both npm and PyPI ecosystems. EXEC Executive Overview

Covering intelligence reported between 06:00 and 10:00 UTC on 2026-05-12, this digest examines three major developments shaping the current threat landscape: a large-scale vendor compromise and extortion agreement involving Instructure and the ShinyHunters group, a rapidly evolving supply chain worm targeting developer ecosystems, and a comprehensive assessment of the 2026 ransomware industry.

The most prominent incident involves Instructure, which has reached a financial agreement with the ShinyHunters extortion group to prevent the public release of approximately 3.65 terabytes of stolen data. The attack exploited multiple cross-site scripting vulnerabilities within Canvas’s Free-for-Teacher environment, granting initial access that enabled the exfiltration of roughly 275 million records affecting nearly 9,000 educational organizations. While Instructure confirmed that course content, user credentials, and submissions remain secure, the compromised dataset contains sensitive institutional information that threat actors could leverage for targeted phishing campaigns. In response, the vendor disabled the affected environment, revoked privileged credentials, and rotated internal cryptographic keys. The FBI has reiterated that ransom payments do not guarantee data non-disclosure or future extortion cessation.

In the software supply chain sector, TeamPCP’s Mini Shai-Hulud campaign has compromised multiple npm and PyPI packages, including libraries from TanStack, Mistral AI, and Guardrails AI. The campaign deploys a self-spreading worm that hijacks GitHub Actions workflows and exchanges OpenID Connect tokens for per-package publish tokens to bypass authentication controls. Compromised repositories are used to deploy credential stealers that exfiltrate sensitive data to external domains while injecting malicious workflow files to serialize repository secrets. Notably, the malicious mistralai PyPI package contains country-aware logic designed to avoid execution in Russian-language environments and carries a one-in-six probability of executing a destructive rm -rf / command on systems located in Israel or Iran.

Broader industry analysis indicates a structural shift in ransomware operations throughout 2026. Ransomware families such as PE32 are increasingly integrating post-quantum cryptography, specifically the NIST-selected Kyber1024 algorithm, to ensure resilience against both classical and quantum decryption efforts. Financially, ransom payments have declined to 28% of incidents, prompting operators to pivot toward encryptionless extortion models that prioritize data theft and public disclosure over file encryption. Initial access remains highly industrialized through broker networks, with RDWeb portals emerging as a preferred remote access vector. Defense evasion has matured into a standard operational phase, with threat actors routinely deploying EDR killers and exploiting signed drivers via Bring Your Own Vulnerable Driver techniques to neutralize endpoint protections. Despite law enforcement takedowns of underground forums and leak sites, the ransomware ecosystem remains operationally stable, with emerging groups like Qilin and The Gentlemen rapidly filling operational vacuums.

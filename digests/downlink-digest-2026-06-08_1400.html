  • CVE-2026-34909 is a path traversal vulnerability that can expose files on the underlying operating system.
  • CVE-2026-34910 is a command injection flaw that can be exploited to execute commands on affected devices.

Technical Root Cause

The root cause of the authentication bypass is a mismatch between request validation and request routing. UniFi OS validates incoming requests using the raw request URI. Nginx routes requests based on a normalized version of the same URI.

This mismatch allows crafted requests to appear, in raw form, as if they are targeting an authentication-exempt endpoint. After normalization, however, the same requests resolve to protected internal routes that should not be publicly accessible. This enables attackers to bypass authentication and reach backend services.

Remote Code Execution and Privilege Escalation

After bypassing authentication, attackers can target a package-update endpoint affected by CVE-2026-34910. The endpoint passes unvalidated user input into a shell command. This behavior allows arbitrary command execution on the system.

The injected commands initially execute under a highly privileged service account. Bishop Fox found that this account has passwordless sudo access to several system binaries. As a result, escalation to root is described as trivial.

According to Bishop Fox, no credentials, user interaction, or prior access are required to obtain a root shell on the target. The researchers validated the RCE chain but did not share full details or a working proof of concept.

Impact and Significance

Bishop Fox emphasized that a UniFi OS Server is not a generic Linux box. It is the management plane for an organization’s network. In deployments where related devices are used, it may also manage physical-access doors, surveillance cameras, and the identities tied to them.

The article quotes Bishop Fox as saying that root access on the appliance is administrative control over everything the console governs. This makes the chain particularly significant because compromise of the server can affect more than the server itself.
